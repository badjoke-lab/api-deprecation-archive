# Disclaimer

API Deprecation Archive summarizes public-source information about developer-facing API lifecycle changes.

Records may be incomplete, outdated, or affected by later provider updates.

Always verify with the official provider documentation before making production changes.

This archive is not:

- official provider documentation
- legal advice
- security advice
- a complete API reference
- a vulnerability or exploit database
- a guarantee that an API is currently usable

The archive may include uncertainty notes when official documentation is ambiguous or when a deadline has passed but removal status has not been re-verified.
